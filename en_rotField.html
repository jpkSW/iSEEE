Rotating Fields

Rotary field windings usually consist of three coils spatially offset by 120°, which are fed with three currents, each phase-shifted by 120°. Why this is the case and how exactly these conditions must be met can be examined in this animation. In this animation, the current amplitudes, the phase positions of the currents and the phase positions of the currents and also the spatial position of the three windings can be varied.

Start the animation and let the gray pointer make one complete revolution!
Reduce the amplitude for the current in string W (purple slider) to zero!
Can a circular rotating field be generated again by changing the amplitude in the other two strings (U: yellow, V: green)?
Obviously, a circular rotating field cannot be generated with two coils having an spatial offset by 120° from each other and a time-dependent phase shift of 120°. Set the amplitudes for the phases U (yellow) and V (green) back to 1.0 and for the phase W to 0.0. Vary the phase-shift for the current V (green) until a circular rotating field is generated again.
Set the phase-shift for the current V (green) to +120° again. To which value do you now have to set the phase-shift for string U (yellow) so that a circular rotating field is created again? Try it out! (But think about the appropriate value first!).
How must you set the phase-shift for the current U (yellow) so that the gray pointer does not turn counterclockwise but clockwise?
Set the amplitudes for phase U (yellow) and V (green) again to 1.0 and for phase W to 0.0. Set the phase-shift for phase U to 0°, V to +120° and W to -120°. Adjust the position for coil V to 90°. To what value must the phase-shift for phase V be set so that a circular rotating field results again? Verify your assumption!
How must the phase-shift be set so that the gray pointer rotates in the opposite direction?
Now set everything back to the original starting values: all amplitudes to 1.0, the phase-shifts to 0° / +120° / -120° and the coil-positions also to 0° / +120° / -120°! There are several possibilities to change the direction of rotation via the phase-shifts! What do they have in common?
